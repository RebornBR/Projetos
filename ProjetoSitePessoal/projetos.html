Banco de questões

    Descrição do projeto:
    O projeto banco de questões foi idealizado e construído com o objetivo de armazenar as questões utilizadas nas avaliações de autoaprendizagem da faculdade. Esse é um projeto onde pude pôr em prática vários dos conteúdos aprendidos durante o Bootcamp de Java, como: POO, Spring FrameWork, PostgreSQL, Swagger e SQL. Com a ajuda da Thaís Correia, colega de faculdade e Dev Front-End, que criou a interface para o uso da api Back End(deploy no render), inicialmente de forma local para testes e, finalmente, após muitos erros, conseguimos colocar esse projeto no ar.
